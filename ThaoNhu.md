# Một chương trình cơ bản của HTML
**Một chương trình cơ bản trong HTML thường bao gồm:** các thành phần cốt lõi để hiển thị nội dung trên trình duyệt. Nó bao gồm cấu trúc trang HTML cơ bản, các thẻ để hiển thị văn bản, hình ảnh, liên kết, và khả năng kết hợp với CSS và JavaScript để định dạng và thêm tính năng
